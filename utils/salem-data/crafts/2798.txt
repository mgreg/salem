==Info==
*[[#Variable Values|Variable Values]]
*[[#Bonus Values|Bonus Values]]

Please see [[Food]] for more info on generic foodstuffs.  Each food page also includes variable information when appropriate.

Each variable food ingredient has an effect on the variable food you put it in, so long as the ingredient is pure.  For example, a [[Humble Meat Pie]] requires 2 [[Any Raw Meat Cut]]s.  If both are [[Deer]], the Deer effect will be added to the food, if a mix of deer and beaver is used, the food will be generic.

For each variable ingredient, the effect will be a combination of Restoration of a Food Group's effectiveness, and/or a multiplier being applied to one or more Humors.  

The following table details the Reduction or Restoration effect and the Humor multiplier each ingredient adds to the food in it's category.

Each Variable ingredient only has an effect in the foodgroup it is in.  For example, foods in the Game Meats group only receive ingredient bonuses from the Game Meat variable ingredients and from the Bonus ingredients (Garlics and Bell Peppers) if any.  They do not receive ingredient bonuses from the Berry variable ingredients, even if a berry is included in the recipe.

*Regarding Corn: Corn is variable when it is [[Corn Oil]], [[Corn Meal]], or [[Any Corn]].
